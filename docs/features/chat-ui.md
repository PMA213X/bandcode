# Chat‑UI 聊天界面模块说明
## 1.模块概述
该模块对应`frontend/src/components/Chat.tsx`，是项目AI对话页面，完成用户输入、消息展示、SSE流式接收AI回复的功能。
## 2.代码结构说明
1. 状态变量
- `isStreaming`：布尔值，标记AI是否正在返回内容；
- `currentMessage`：保存AI实时返回的文字；
- `messages`：数组，存储全部历史聊天记录。
2. 页面区域划分
- 消息列表区域：循环渲染用户消息和AI回复消息；
- 加载提示区域：AI返回内容期间显示“正在处理…”；
- 输入框区域：接收用户输入内容，做非空判断，禁止发送空白内容。
## 3.核心技术
1. React+TSX语法编写组件；
2. 借助SSE（Server‑Sent‑Events）长连接，后端逐字推送AI回答，实现打字机效果；
3. 配合全局Layout布局组件，页面自适应窗口大小。
