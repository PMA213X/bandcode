# SimpleCoder Agent（简单编码智能体）

## 基本信息

| 属性 | 值 |
|------|-----|
| Agent名称 | SimpleCoder Agent |
| 中文名称 | 简单编码智能体 |
| 类型 | 业务级，子Agent |
| 模型 | mimo-v2.5 |
| 温度 | 0.2 |

## 适用场景

- UI修改：样式调整、布局优化、文本修改
- 配置调整：配置文件参数修改
- 单文件修改：单个文件内的代码变更
- 小型Bug修复：简单的逻辑修复、拼写错误等

## 工具权限

| 工具 | 权限 |
|------|------|
| read | allow |
| edit | allow |
| bash | allow |

## 自动升级条件

当任务超出SimpleCoder处理能力时，自动升级至ComplexCoder：

- 超过2个文件需要修改
- 超过300行代码变更
- 涉及架构调整

## 输出格式

完成编码后输出修改结果，包含变更文件列表和变更说明。
